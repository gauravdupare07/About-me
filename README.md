# 🌐 Personal Portfolio Website

A responsive personal portfolio website built using **HTML, CSS, and JavaScript**.  
This project showcases my **skills, projects, and contact information** in a modern, clean, and mobile-friendly design.

---

## ✨ Features
- Responsive design (works on desktop & mobile 📱💻)
- Smooth scrolling navigation
- About Me section with profile and interests
- Projects section with interactive cards & GitHub links
- Contact section with email and social links
- Modern UI with animations and hover effects
